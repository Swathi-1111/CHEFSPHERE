# 📖 Project Overview

CHEFSPHERE is a modern Android application developed to help users
discover recipes that match their personal health goals, dietary
preferences, allergies, cooking skills, and nutritional requirements.

Unlike traditional recipe applications, CHEFSPHERE focuses on
**personalization**. During onboarding, users create a detailed health
profile that is used to tailor recipe suggestions and improve the
overall cooking experience.

------------------------------------------------------------------------

# 🎯 Objectives

-   Deliver personalized recipe recommendations.
-   Store user health and dietary profiles.
-   Improve healthy eating habits.
-   Provide an intuitive Android experience using Jetpack Compose.
-   Demonstrate modern Android development practices.

------------------------------------------------------------------------

# ❗ Problem Statement

Many recipe applications provide the same recommendations to every user
without considering health conditions, allergies, dietary preferences,
cooking skills, or fitness goals.

CHEFSPHERE addresses this challenge by building a personalized profile
for each user and using it to support healthier meal decisions.

# 🔄 Application Workflow

1.  User registers or logs in.
2.  User completes the health profile.
3.  Profile is stored in SQLite.
4.  Recipes are displayed based on stored preferences.
5.  User explores recipes.
6.  Cooking history is saved for future reference.
